O que é HTML?

HTML é a linguagem base dos websites. Ela serve para nós estruturarmos textos, criar links, exibir imagens, vídeo, etc.O seu significado é HyperText Markup Language, traduzindo para português, Linguagem de Marcação de Hipertexto.

  • HyperText: significa hypertexto, um texto que contém, refere-se aos links que conectam páginas da Web entre si.
  • Markup: significa a marcação desse texto. Geralmente fazemos isso com o uso de uma tag, ou elemento HTML. (entederemos isso em breve)

Vale ressaltar que HTML não é uma linguagem de programação, já qque seu objetivo não é criar um software, e sim, marcar a estrutura de linguagens de programação, como por exemplo o JavaScript. Em resumo, o HTML indicará onde os elementos aparecerão na aplicação.

Existem inúmeras tags e cada uma delas irá servir para um determinado propósito. Exemplo: Imagem, texto grande, links, parágrafo, título,etc. Para cada uma dessas palavras que trouxe, existe uma tag se eu quiser inserir algo do tipo no meu website.

Sintaxe de uma tag

As tags também possuem atributos que adicionam informações e configurações a uma tag. Por exemplo o elemento "a" (ou âncora), ela possui um atributo chamado href, onde cria-se uma hiperligação nas páginas web, arquivos.

Atributos

Os atributos representam caracteristícas e funcionalidades de cada elemento HTML. Temos os atributos globais, como id ou style, e também temos os específicos como o alt que só encontramos no elemento img.
