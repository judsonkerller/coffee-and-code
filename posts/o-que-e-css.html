O que é CSS?

Ele é a apresentação visual para o cliente. Ele estiliza o HTML. O Acrônimo de CSS é Cascade Style Sheets, ou seja, folha de estilo em cascata. Enquanto que o HTML, nós estruturamos os textos, imagens, etc. Com o CSS nós estilizamos o que criamos com o HTML.
